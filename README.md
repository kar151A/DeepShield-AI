# 🛡️ DeepShield AI

An AI-powered system to detect deepfake images/videos and generate forensic evidence reports.

---

## 🚀 Features

* Upload image/video for analysis
* Detect Real vs Fake using AI
* Confidence score & explanation
* Generate downloadable forensic report (PDF)
* Visual indicators of manipulation

---

## 🧠 Tech Stack

* Frontend: React + Tailwind
* Backend: FastAPI
* AI: Deep Learning / OpenCV
* Deployment: Vercel + Render

## ⚙️ How to Run

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
