# 🛡️ DeepShield AI

DeepShield AI is a web-based application that detects fake or manipulated images using computer vision techniques.

## 🚀 Features
- Upload image for analysis
- Detect FAKE vs REAL
- Confidence score
- Visual output preview
- Responsive UI with loader

## 🧠 Tech Stack
- Backend: FastAPI
- Frontend: React (Vite)
- Image Processing: OpenCV

## ▶️ Run Locally

### Backend
cd backend  
uvicorn main:app --reload  

### Frontend
cd frontend/vite-project  
npm install  
npm run dev
